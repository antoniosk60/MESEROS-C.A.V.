# Galería (imágenes)

Coloca aquí tus fotos reales para que el sitio las muestre automáticamente.

En este repo ya se “sembraron” imágenes desde `data/img/` y se copiaron con nombres cortos para el sitio.

Archivos esperados por el proyecto:

- `hero.jpg`
- `avatar.jpg`
- `eventos-sociales.jpg`
- `corporativos.jpg`
- `cenas-privadas.jpg`
- `01.jpg` a `18.jpg`

Puedes usar `.jpg`, `.jpeg` o `.png`, pero si cambias extensiones/nombres actualiza `data/gallery.ts`.
